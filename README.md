# Smart Home Intrusion Detection System

## Project Overview
This project is an AI + IoT based smart security system developed using Python and OpenCV.

The system monitors live camera feed and detects human presence in real time. If an unknown person is detected, the system captures the image, stores logs, and sends an email alert.

## Features
- Real-time Human Detection
- Known / Unknown Person Detection
- Email Notification
- Image Capture with Timestamp
- CSV Event Logging
- Smart Security Monitoring

## Technologies Used
- Python
- OpenCV
- Streamlit
- SMTP Email Service
- CSV File Handling

## Files
- app.py → Hosted web interface
- smart_intrusion_demo.py → Main execution file
- requirements.txt → Libraries required
